Explain the concept of Responsive Design. What does the term mean?

The concept of responsive design refers to the way a website or application's design responds to its environment.

How does responsive design relate to web technologies?

Web pages were previously built for screen size. As more users began to browse websites using cellphones,
Contents of the web pages were inaccessible as it were too large for the screen.
Contents of the websites were not able to adapt to different display sizes.
Responsive design relate to web technologies through HTML and CSS.
Responsive web design ensure that regardless of the screen size, the web page must be
usable and visually appealing; this means users should be able to browse
on a web page using various devices such as laptop, tablet and smartphones.

What did you learn about building web apps for different devices through your research?

Based on my research, I learned that I should always keep in mind that various devices will be used to access the web application.
I should always test to see if the web application is easy to use not just in desktop, but in other devices as well such as smartphone and tablet.
Furthermore, the web application should be readable regardless of the display siz; this means I should use proper font style, size, and colour for readability.
